Bedankt, open source helden

Deze website was er niet geweest zonder open source software. Het bouwen van software is geen koud kunstje. Daarom bedankt ik graag de ontwikkelaars die hun handige software vrij beschikbaar hebben gesteld.

Hugo: snelle en krachtige generator

Deze hele website is statisch en gegenereerd met Hugo . Statisch websites zijn een stuk sneller dan een dynamische site, omdat bezoekers vooraf gegeneerde html-pagina’s te zien krijgen. Dit is anders dan dynamische system zoals Wordpress. Dynamische websites genereren een pagina namelijk pas als een bezoeker de pagina opvraagt (en doen dit iedere keer opnieuw). Voor een eenvoudig blog is dit overbodig.

Het is geen feest om vanaf de grond af zelf een website te maken. Hugo maakt het onderhouden van een statische website gelukkig erg makkelijk. Artikelen schrijf ik op mijn eigen computer en met één druk op de knop wordt een nieuwe site gegenereerd.

Beautifulhugo: basis voor het ontwerp

Het ontwerp van deze website is gebaseerd op beautifulhugo . Je ziet hier een aangepaste versie maar een groot deel kon ik hergebruiken.

Isso: privacy-vriendelijke reacties

Een blog is natuurlijk niets zonder de mogelijkheid om te reageren. De meeste blogs gebruiken Disqus als reactiesysteem. Helaas is dit systeem niet goed voor de privacy van je bezoekers. Ik gebruik daarom Isso ; een simpel maar eenvoudig alternatief dat reacties opslaat op je eigen server.
